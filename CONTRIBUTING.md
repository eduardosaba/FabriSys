# Contribuindo para o FabriSys

Obrigado por querer contribuir! Algumas diretrizes rápidas:

1. Fork e clone o repositório.
2. Crie uma branch com o padrão `feature/descricao` ou `fix/descricao`.
3. Antes de commitar, rode:

```bash
pnpm lint
pnpm test --if-present
pnpm format
```

4. Faça PR contra a branch `main` e use o template de PR.
5. Seja claro nas mensagens de commit e na descrição do PR.

Para dúvidas, abra uma issue no repositório.
